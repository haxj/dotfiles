# Dotfiles

## Install

### inputrc

```
cp inputrc ~/.inputrc
```

### tmux

```
cp tmux.conf ~/.tmux.conf
```

### NvChad 2.5

```
mv ~/.config/nvim ~/.config/nvim_backup
mv ~/.local/share/nvim ~/.local/share/nvim_backup
cp -r nvim_nvchad_2.5 ~/.config/nvim
```

### i3status-rust v0.33.2

```
cp -r i3status-rust_v0.33.2 ~/.config/i3status-rust
```
